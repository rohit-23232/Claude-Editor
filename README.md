# AI Editor

Modern AI Assistant powered by:

- Claude API
- OpenAI GPT
- Google Gemini

## Features

- Modern UI
- Multi AI Providers
- Markdown
- Chat History
- Dark / Light Mode
- Voice Input
- Responsive Design
- Export Chats
- Local Storage

## Installation

```bash
npm install
```

Run

```bash
npm run dev
```

Open

```
http://localhost:3000
```

## Environment

Create

```
.env
```

Copy from

```
.env.example
```

Add

- ANTHROPIC_API_KEY
- OPENAI_API_KEY
- GEMINI_API_KEY
